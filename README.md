# 프로젝트명

## 실행 방법

npm install
npm run dev

## 기술 스택

- React 19.2.0
- TypeScript 5.9.3
- Vite 7.3.1 (Build Tool)
- ESLint + Prettier (Code Quality)
- Airbnb ESLint Style Guide
